# HPUPM Moodle Docker


## Starting the Stack

### 1. Populate .env File

Copy the `.env-sample` file and rename the new file as `.env`. Then, populate the `.env` file with correct values.

### 2. Start the Stack

To run the Moodle stack, use the following command:

```bash
docker-compose up -d
```

## Stopping the Stack

```bash
docker-compose down
```
